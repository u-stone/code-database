目录说明
 .\TMG_Worker_Photo\ 	存放的是员工的照片，目前只支持jpg格式的图片，图片的大小最好统一。
.\music\			存放的是程序使用的音频文件，是用来作背景音乐的
.\init.ini			该文件是整个程序的配置文件，可以配置抽奖的信息，比如一共几等奖，每等奖多少奖品，什么奖品
.\bg.bmp			这个是程序的背景图片
.\frontpage.bmp		这个是程序启动之后第一个显示的图片
.\data.txt			这个是原来使用手机号抽奖的数据存放处，现在弃用。
.\Lottery.exe		这个是抽奖的主程序
其他几个是运行时dll文件.

可以通过替换相应的文件来配置出不同的界面，通过改写init.ini文件来得到不同的抽奖过程。
年会开始之前最好先在相应的计算机上测试一下程序，有问题，先解决。
